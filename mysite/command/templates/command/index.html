기본기능

/명령어

  • 명령어 목록을 봅니다. /도움말 로도 호출가능합니다.

/패치노트

  • 패치노트를 열람할 수 있습니다.

/안녕

  • 퓨이린 봇이 인사를 합니다.

/다

  • 뀨다뀨

/피망

  • 퓨이린 봇에게 피망을 줄 수 있습니다. 매일 한 개씩 주도록 합시다.

/청소 (메시지 개수)

  • 메시지를 입력한 개수만큼 삭제합니다.
    • 변수 (정수) 하나를 필요로 하는 명령어입니다.
      • ex) /청소 5
    • 필요권한
      채널관리 권한이 필요합니다.
    • 주의
      너무 큰 숫자를 입력할 경우 봇의 성능이 급격하게 저하됩니다.

/출첵

  • 출석체크 현황을 확인합니다. /출석체크 로도 호출할 수 있습니다.

/나무

  • 봇이 나무를 캐줍니다. 잡기능

/설정

  • 서버 별로 설정 가능한 퓨이린 봇의 설정을 관리할 수 있습니다.
    • 참고
      v.10.4 기준 현재 음성 채팅 참여 알림 설정이 가능합니다.

편의기능

/뽑기 (숫자1) (숫자2)

  • 입력한 두 정수 사이의 수를 임의로 추출합니다. 이 때 추출되는 수는 양끝값을 포함합니다.
    • 변수(정수) 두 개를 필요로하는 명령어입니다.
      • ex) /뽑기 1 10

/골라

  • 추첨 시스템을 불러옵니다.
    • 예시
    • 파생기능 /골라` 목록1 목록2 ...
      • 하나의 목록만 추출할 때 사용할 수 있습니다.
      • ex) /골라` 코카콜라 펩시

/코로나

  • 봇이 코로나 관련 정보를 제공합니다.

/날씨 (지역)

  • 봇이 입력된 지역의 날씨 정보를 제공합니다.
    • 변수(문자열) 하나를 필요로하는 명령어입니다.
      • ex) /날씨 서울 서초구

/이미지 (검색어)

  • 봇이 구글링을 통해 얻은 사진 중 아무거나 랜덤으로 전송합니다.
    • 변수(검색어) 하나를 필요로하는 명령어입니다.
      • ex) /이미지 고양이
    • 검색된 사진으로 인해 발생한 피해는 봇이 책임지지 않습니다.

미니게임

/랜덤게임

  • 랜덤게임을 시작합니다. 봇이 제공 게임 중 하나를 무작위로 불러옵니다.
    • 현재 제공 게임 : 끝말잇기, 업다운, 블랙잭, 가위바위보
    • 각 게임은 명령어를 통해 따로 불러올 수 있습니다.
      • ex) /끝말잇기 /업다운 /블랙잭 /가위바위보

/배라

  • 배스킨라빈스 31 게임을 시작합니다.
    • 필요권한
      역할관리와 채널관리 권한이 필요합니다.

게임 관련 기능

/롤전적 (소환사명)

  • 봇이 입력된 소환사의 전적 관련 정보를 제공합니다.
    • 변수(문자열) 하나를 필요로 하는 명령어입니다.
      • ex) /롤전적 퓨이린

/롤승률 (소환사1, 소환사2, ... , 소환사5)

  • 봇이 입력된 소환사 5명의 승률을 분석해 예상 승률을 제공합니다.
    • 예시

/메이플

  • 메이플 편의기능을 불러옵니다.

강화(인챈트) 기능

/강화 (멤버명)

  • 입력된 멤버를 인챈트합니다.
    • 변수(문자열) 하나를 필요로 하는 명령어입니다.
      • ex) /강화 퓨이린

/강화 (멤버명)

  • 입력된 멤버를 인챈트합니다.
    • 변수(문자열) 하나를 필요로 하는 명령어입니다.
      • ex) /강화 퓨이린
    • 참고
      서버 내에서 변경된 닉네임은 봇이 인식하지 못합니다. 이 경우 /강화랭킹 을 통해 조회되는 닉네임을 볼 수 있습니다.

/강화현황 (멤버명)

  • 입력된 멤버의 인챈트 단계를 조회합니다.
    • 변수(문자열) 하나를 필요로 하는 명령어입니다.
      • ex) /강화현황 퓨이린

/강화랭킹

  • 서버 내의 인챈트 랭킹을 조회합니다.

음악 기능

/커몬

  • 봇이 음성채팅방에 참여합니다.
    • 알림
      해당 명령어를 사용한 멤버가 음성채팅방에 들어와 있어야 합니다.

/바이

  • 봇이 음성채팅방에서 나갑니다.

/재생 (유튜브 링크)

  • 입력된 링크의 음악을 재생합니다.
    • 변수(유튜브 링크) 하나를 필요로 하는 명령어입니다.
      • ex) /재생 https://www.youtube.com/watch?v=gdZLi9oWNZg
    • 주의
      음악이 재생되는데 다소 시간이 소요됩니다. (10초 내외) 재생 명령어를 중복 사용하지 않도록 신경 써주시기 바랍니다. (request limit 관련)
    • 참고
      재생목록은 재생할 수 없습니다.

/일시정지

  • 현재 재생중인 곡을 일시정지합니다.

/재개

  • 일시정지된 곡을 다시 재생합니다.

/정지

  • 현재 재생중인 음악을 완전히 정지합니다.
